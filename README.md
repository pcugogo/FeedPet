## 'FeedPet'?
* `댕댕이`와 `냥냥이`들의 반려가족인 반려인들의 편의를 제공하이 위한 앱!
* 우리 아이들의 사료구입시 찾게 되는 정보를 얻고 성분을 알려줘요
* 검색과 기능성 필터를 활용하여 쉽게 검색할수 있어요.
* 알림을 설정하여 사료주는 시간을 알림 받을수 있어요.
